# ts-boiler

A basic project scaffolding for typescript projects

## Usage

```
npm run dev
```

Visit local site on http://localhost:4444

## Features

-   Compiles TypeScript
-   Imports `.shader`, `.vert` and `.frag` files as strings
-   Live reloading on source code change
